## Production Notes

### Visual Style:
- **Color scheme:** Match the site's dark, high-tech theme
- **Transitions:** Smooth fades and quick cuts
- **Text overlays:** Use the site's font (modern, techy)
- **Animations:** Subtle zoom-ins on key features

### Screen Recording Tips:
1. Use 1920x1080 resolution
2. Record at 60fps for smooth animations
3. Use cursor highlights/animations
4. Add subtle zoom effects on important elements
5. Keep transitions fast (0.5-1 second)

### Voiceover:
- **Tone:** Energetic, confident, tech-forward
- **Pace:** Fast but clear (150-160 words/min)
- **Music:** Upbeat electronic/tech background (low volume)

### Key Moments to Capture:
1. Dashboard load animation
2. Scrolling through news feed
3. Clicking on video idea cards
4. n8n workflow visualization
5. Real-time data updates
